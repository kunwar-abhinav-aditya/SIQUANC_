Have you got multiple questions to ask at once?
No worries, this module helps exactly with that!
Select a valid csv file from your local filesystem and upload it. Then give us some time to generate the results and give them back to you. The time will vary based on the number of questions in the file.
Each entry in the file should follow the following format
question-id,question
Although one question could be repeated several times in a file (though we wonder why you would do that), every question-id must be unique for getting the correct results.
So there you go!!
