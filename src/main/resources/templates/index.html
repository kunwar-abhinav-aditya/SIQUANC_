THEME

Welcome to Frankenstein!


Frankenstein is a platform that seamlessly integrates heterogeneous components performing different tasks of a Question-Answering System (e.g. entity recognition, relation linking). These components can either be used independently for annotating any short text or as part of a QA pipeline (i.e. system).
